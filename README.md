# Backend MongoDB Authentication Project

## What This Is

A simple Node.js backend with user registration that connects to MongoDB Atlas.

## What I Did

1. Created MongoDB Atlas account and cluster
2. Connected the app to my database
3. Tested user registration endpoint
4. Users can register with name, email, and password

## How to Run
```bash
npm install
cd src
node server.js
```

## Tech Used

- Node.js
- Express
- MongoDB
- Mongoose
- bcrypt
